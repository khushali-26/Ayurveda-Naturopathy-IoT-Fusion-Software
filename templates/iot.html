Working Principle

In the proposed system, IoT sensors continuously collect physiological and lifestyle-related data such as heart rate, temperature, sleep quality, and physical activity. The data is transmitted through an ESP32 microcontroller to the Flask application where it is stored and analyzed. The wellness engine combines Ayurvedic and Naturopathic principles to generate personalized wellness recommendations for the user.

Current Project Status

The present implementation demonstrates the software prototype. The current implementation functions as a web-based wellness assessment system developed using Python, Flask, HTML, CSS, and SQLite. Users can register, complete a comprehensive Ayurvedic and Naturopathic questionnaire, receive personalized dosha analysis (Vata, Pitta, and Kapha), view wellness scores, obtain customized recommendations, and store assessment records in a database for future reference.

The system successfully demonstrates the integration of traditional Ayurvedic and Naturopathic wellness concepts with modern web technologies. IoT sensor integration is proposed as a future enhancement to enable real-time health monitoring and automated wellness analysis and can be implemented using ESP32-based sensor modules.
